배열 데이터 추가 / 삭제 / 정렬

1. 배열의 끝에서 데이터 추가
배열.push(추가데이터);

2. 배열의 중간에서 데이터 추가
배열.splice(지정위치, 0, 추가데이터);

3. 배열의 중간에서 데이터 삭제
배열.splice(지정위치, 1);
**splice(index, 삭제갯수, 추가데이터)

4. 배열의 중간에서 데이터 변경
// javaScript에서는 변경할 데이터 인덱스에 새로운 값을 추가하면 되지만, Vue.js는 값이 바뀐 것을
인지할 수 없게 되어 splice 메소드를 사용합니다.
배열.splice(지정위치, 1, 변경데이터);
