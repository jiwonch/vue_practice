§transition
애니메이션 효과를 내고 싶을 때
1. 나타날(혹은 사라질) HTML 태그를 transition 태그로 감싼다.

<transition>
   <div v-if="isOK">표시/비표시의 변경</div>
</transition>

-------------------------------------------------------------------------------------------
2. 어떻게 변화할지를 CSS로 준비한다.

태그가 나타날 때
   .v-enter : 나타나기 전의 상태
   .v-enter-active : 나타나고 있는 상태
   .v-enter-to : 나타난 상태  => 나타난 상태는 기본 상태이므로 설정하지 않음.

태그가 사라질 때
   .v-leave : 사라지기 전의 상태
   .v-leave-active : 사라지고 있는 상태
   .v-leave-to : 사라진 상태

태그가 이동할 때
   .v-move : 태그가 이동할 때


---------------------------------------------------------------------------------------------
§리스트 트렌지션 리스트 애니메이션 : trnasition-group

리스트의 수가 증감하거나 위치가 이동할 때 애니메이션 효과는 transition-group 태그로 감쌉니다.

<transition-group>
   <div v-for="item in dataArray" v-bind:key="item">{{item}}</div>
</transition-group>
